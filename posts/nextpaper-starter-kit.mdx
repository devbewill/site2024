NextPaper is a minimal, responsive and SEO-friendly NextJs blog theme. This theme is designed and crafted based on my personal site and built using Nextjs 14, tailwind and MDX.

If you like it and you want use it for your personal site, blog or portfolio you can do it for free.

## ⭐️ How to get it

Simply clone the [repository](https://github.com/devbewill/nextpaper-starter-site/) and deploy on your favorite hosting platform (Vercel, Netlify or AWS Amplify for example).

## 🔥 Features

- [x] Nextjs 14 with App router and SSG
- [x] Tailwind
- [x] Markdown (MDX Ready)
- [x] responsive (mobile ~ desktops)
- [x] SEO-friendly
- [x] Blog and Agenda
- [x] Metadata ready
- [x] Simple, light and clean UI

## 📖 Installation

```bash
git clone [this_repo_url] your_app_folder
cd your_app_folder
npm install
npm run dev
```
